# Luxe Beauty - موقع تجارة إلكترونية للمنتجات التجميلية

## الملفات الرئيسية

### صفحات العملاء:
- `index.html` - الصفحة الرئيسية
- `products.html` - صفحة المنتجات مع الفلاتر
- `about.html` - صفحة من نحن
- `cart.html` - سلة التسوق
- `checkout.html` - صفحة الدفع
- `order-success.html` - صفحة نجاح الطلب
- `wishlist.html` - قائمة الأمنيات
- `profile.html` - لوحة تحكم المستخدم
- `login.html` - تسجيل الدخول والتسجيل

### صفحات الإدارة:
- `admin-login.html` - تسجيل دخول المدير
- `admin-dashboard.html` - لوحة تحكم المدير

### ملفات JavaScript:
- `app.js` - الوظائف المشتركة (السلة، القائمة المفضلة، الثيم)
- `checkout.js` - وظائف صفحة الدفع
- `admin-script.js` - وظائف لوحة تحكم المدير

## المميزات

✅ وضع الليل/النهار (Dark/Light Mode)
✅ سلة التسوق
✅ قائمة الأمنيات
✅ نظام المستخدمين
✅ لوحة تحكم المدير
✅ صفحة الدفع الكاملة
✅ تتبع الطلبات

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. تصفح المنتجات
3. أضف المنتجات إلى السلة
4. اذهب إلى صفحة الدفع
5. أكمل عملية الشراء

## ملاحظات

- جميع البيانات محفوظة في localStorage
- الموقع يعمل بدون backend
- جميع الصفحات responsive ومتوافقة مع الموبايل
