## 📦 نظام الطلبات

### هيكل الطلب
```javascript
{
    orderId: 'ORD-1234567890',
    userId: 'customer@email.com',
    customerInfo: {
        name: 'أحمد محمد',
        email: 'customer@email.com',
        phone: '+20123456789',
        address: { street, apartment, city, state, zip, country }
    },
    items: [
        { productId, name, price, quantity, image }
    ],
    payment: {
        method: 'card',  // card, wallet, cod
        status: 'completed',
        transactionId: 'TXN-1234567890'
    },
    pricing: {
        subtotal: 90.00,
        shipping: 0,
        tax: 9.00,
        total: 99.00
    },
    status: 'pending',  // pending, processing, shipped, delivered, cancelled
    statusHistory: [
        { status, timestamp, note }
    ],
    createdAt: '2026-03-13T10:30:00Z',
    updatedAt: '2026-03-13T10:30:00Z'
}
```

### حالات الطلب
- **Pending** - في انتظار المعالجة
- **Processing** - قيد التجهيز
- **Shipped** - تم الشحن
- **Delivered** - تم التوصيل
- **Cancelled** - ملغي
