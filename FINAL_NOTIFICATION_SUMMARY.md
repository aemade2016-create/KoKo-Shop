## 🔄 كيف يعمل النظام

### عند إلغاء طلب من العميل:
```
العميل → Cancel Order
    ↓
تحديث حالة الطلب → cancelled
    ↓
إشعار للأدمن ← "Customer [Name] cancelled order #[ID]"
    ↓
إشعار للعميل ← "Your order has been cancelled successfully"
    ↓
تحديث Badge للأدمن
```

### عند تحديث حالة الطلب من الأدمن:
```
الأدمن → Update Order Status
    ↓
تحديث حالة الطلب → [New Status]
    ↓
إشعار للعميل ← "Your order status has been updated to [Status]"
    ↓
تحديث Badge للعميل
```

## 🧪 اختبار النظام

### الخطوات:

1. **اختبار إلغاء الطلب:**
   - سجل دخول كعميل
   - اذهب إلى Profile > My Orders
   - اضغط Cancel Order على طلب
   - تحقق من الإشعار
   - سجل دخول كأدمن
   - تحقق من ظهور الإشعار

2. **اختبار تحديث الطلب:**
   - سجل دخول كأدمن
   - اذهب إلى Orders Tab
   - غير حالة طلب
   - سجل دخول كالعميل
   - تحقق من ظهور الإشعار

3. **اختبار Badge:**
   - تحقق من ظهور Badge عند وجود إشعارات غير مقروءة
   - تحقق من اختفاء Badge بعد قراءة جميع الإشعارات
