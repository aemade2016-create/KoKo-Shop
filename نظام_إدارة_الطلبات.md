```javascript

        subtotal: 99.98,
        shipping: 5.99,
        tax: 10.00,

    statusHistory: [

    updatedAt: '2026-03-13T10:30:00.000Z'
```

## 🔄 دورة حياة الطلب (Order Lifecycle)

### 1. إنشاء الطلب (Order Creation)
**الملف:** `checkout.js` - دالة `createOrder()`

```javascript
function createOrder(formData) {
    var cart = getCart();
    var subtotal = 0;

    // حساب المجموع الفرعي
    cart.forEach(function (item) {
        var price = parseFloat(item.price) || 0;
        var quantity = parseInt(item.quantity) || 1;
        subtotal += price * quantity;
    });

    // حساب الشحن والضريبة
    var shipping = subtotal > 50 ? 0 : 5.99;
    var tax = subtotal * 0.1;
    var total = subtotal + shipping + tax;

    // إنشاء كائن الطلب
    var order = {
        orderId: 'ORD-' + Date.now(),
        userId: getCurrentUser() ? getCurrentUser().email : formData.email,
        customerInfo: { /* ... */ },
        items: cart.map(/* ... */),
        pricing: { subtotal, shipping, tax, total },
        payment: { /* ... */ },
        status: 'pending',
        statusHistory: [{ /* ... */ }],
        createdAt: new Date().toISOString(),
        updatedAt: new Date().toISOString()
    };

    return order;
}
```

### 2. حفظ الطلب (Order Storage)
**الملف:** `checkout.js` - دالة `saveOrder()`

يتم حفظ الطلب في 3 أماكن:

#### أ) طلبات المستخدم (User Orders)
```javascript
localStorage.setItem('luxe_orders', JSON.stringify(orders));
```

#### ب) طلبات الأدمن (Admin Orders)
```javascript
localStorage.setItem('luxe_admin_orders', JSON.stringify(adminOrders));
```

#### ج) تحديث بيانات المستخدم (User Profile)
```javascript
// إضافة الطلب لسجل المستخدم
users[userIndex].orders.push(order.orderId);
users[userIndex].totalSpent += order.pricing.total;
```

## 📊 حالات الطلب (Order Status)

| الحالة | الوصف | اللون |
|--------|-------|-------|
| `pending` | قيد الانتظار - تم استلام الطلب | أصفر |
| `processing` | قيد المعالجة - جاري تجهيز الطلب | أزرق |
| `shipped` | تم الشحن - الطلب في الطريق | بنفسجي |
| `delivered` | تم التوصيل - وصل للعميل | أخضر |
| `cancelled` | ملغي - تم إلغاء الطلب | أحمر |
