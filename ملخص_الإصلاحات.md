**المنتج 2: مرطب مائي**

- الزر: `onclick="addToCartBestSeller(2)"`

**المنتج 3: واقي شمس SPF 50**
- معرف المنتج: 8
- السعر: 32.99 جنيه مصري
- الزر: `onclick="addToCartBestSeller(8)"`

### دالة جديدة في `app.js`:
```javascript
function addToCartBestSeller(productId) {
    // الحصول على المنتجات من localStorage
    var products = JSON.parse(localStorage.getItem('luxe_products')) || [];
    var product = products.find(function(p) { return p.id === productId; });
    
    if (product) {
        addToCart(productId, {
            name: product.name,
            price: product.price,
            image: product.image
        });
    } else {
        showToast('المنتج غير موجود', 'error');
    }
}
```
